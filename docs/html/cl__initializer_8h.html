◆ CREATE_BINARY

#define CREATE_BINARY   0 >> 1

A macro used in initCL.

This macro specifies that a binary file of the name specified should be created using the source code at the provided program source. It also creates a program with the source. This macro excludes LOAD_FROM_SOURCE and LOAD_FROM_BINARY

◆ LOAD_FROM_BINARY

#define LOAD_FROM_BINARY   4 >> 1

A macro used in initCL.

This macro specifies that the program should be made using a binary from the specified binary file. This macro will not work if CREATE_BINARY is specified and it will exclude LOAD_FROM_SOURCE

◆ LOAD_FROM_SOURCE

#define LOAD_FROM_SOURCE   3 >> 1

A macro used in initCL.

This macro specifies that the program should be made using source code from the specified program source file and it shouldn't create a binary. This macro will not work if CREATE_BINARY is specified or LOAD_FROM_BINARY is specified

◆ OUT_OF_ORDER

#define OUT_OF_ORDER   2 >> 1

A macro used in initCL.

This macro specifies that the OpenCL command queue should be an out of order queue.

◆ PROFILE

#define PROFILE   1 >> 1

A macro used in initCL.

This macro specifies that the OpenCL command queue created should have profiling enabled
